# js-project-template
Basic starting point for my JS projects

### Using this template
A few ways to leverage this template:

```shell
$ git clone https://github.com/cjus/js-project-template.git
$ mv js-project-template myproject
$ cd myproject
$ rm -rf .git
$ git init
$ git remote add origin https://github.com/cjus/myproject.git
$ git add .
$ git commit -m "Initial commit"
$ git push -u origin master
```

Other ways:
* Use [Github's Repo Import Tool](https://github.com/new/import) and enter `https://github.com/cjus/js-project-template` in the old repositoryâ€™s clone field.
